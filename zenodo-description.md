A dataset for interfacial electrochemistry data, currently focusing on the cyclic voltammograms for single crystal electrodes recorded in aqueous and non aqueous electrolytes.
The dataset can be browsed on [echemdb.org/cv](https://www.echemdb.org/CV) and consists of published data, which is either retraced from source PDFs or provided by the authors.

This dataset is generated automatically during a release from the input files provided to the [https://github.com/echemdb/electrochemistry-data] repository.

The data is maintained and provided by the [echemdb](https://github.com/echemdb) github community.
